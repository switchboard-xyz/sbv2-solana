### <SiRust className="devicons" /> switchboard-v2

<details><summary><b>switchboard-v2:</b> Rust crate to interact with Switchboard V2 from an on-chain program.</summary>

**Documentation:** [crates.io](https://crates.io/crates/switchboard-v2), [docs.rs](https://docs.rs/switchboard-v2)

**Source Code:** [github.com/switchboard-xyz/switchboardv2-rs](https://github.com/switchboard-xyz/switchboardv2-rs)

**Install:**
Add the following line to your Cargo.toml

```bash toml
[dependencies]
switchboard-v2 = "0.1.8"
```

</details>

### <SiTypescript className="devicons"/> <SiRust className="devicons" /> switchboardv2-example

<details><summary><b>switchboardv2-example:</b> Example demonstrating the full path from creating an oracle queue and aggregator, then running an oracle and updating it's result. </summary>

**Documentation:** [github.com/switchboard-xyz/switchboard-v2-example](https://github.com/switchboard-xyz/switchboard-v2-example)

**Source Code: ** [github.com/switchboard-xyz/switchboard-v2-example](https://github.com/switchboard-xyz/switchboard-v2-example)

**Install:** Run the following command

```bash
git clone https://github.com/switchboard-xyz/switchboard-v2-example
cd switchboard-v2-example
npm install
```

</details>
